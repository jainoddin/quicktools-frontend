# QuickTools.ai - Cursor AI Rules

## Persona
You are an expert Next.js and Tailwind CSS developer building a premium AI SaaS platform. 

## General Rules
1. **Always use App Router** (no Pages router).
2. **TypeScript First:** Ensure all props and API responses are typed with interfaces or Zod schemas.
3. **Component Modularity:** Break down large files into small, reusable components. Keep components in the `components/` folder.
4. **Mobile First Design:** Always write Tailwind classes starting with mobile sizes, then scale up using `md:` and `lg:`.
5. **Clean Code:** Remove `console.log` before completing a task. Never push dummy data to production.
6. **No Placeholder Text:** Use real-sounding copy for all UI elements.
