# QuickTools.ai - Project Context

## Overview
QuickTools.ai is a premium AI SaaS platform built by a solo developer. It aims to provide fast, SEO-friendly, and high-performance AI tools to users.

## Initial Launch Strategy
The MVP (Minimum Viable Product) will launch with exactly **5 Core AI Tools**. This ensures high quality, easy maintenance, and focused marketing for a solo developer.

## Core Features
1. **5 AI Tools:** (To be defined in feature-specs, e.g., AI Text Generator, Image Optimizer, Summarizer, etc.)
2. **Automated AI Blog:** A fully autonomous blogging system that uses AI to write and publish daily articles based on updated, trending data. This will drive organic SEO traffic effortlessly.
3. **High Performance:** Page speed must be extremely fast (Lighthouse 95+).
4. **SEO Optimized:** Architecture built for perfect Google crawling and fast indexing.

## Developer Constraints
- **Solo Developer:** Architecture must be simple, highly maintainable, and cost-effective (utilizing free/low-cost tiers for hosting and APIs).
- **Automated Workflows:** Rely heavily on AI and cron jobs to reduce manual work and automate content creation.
